日志跟踪(alog)
==============

日志自动保存
-----------

* 可以将系统产生的日志保存到/sdcard/alog/目录下。
* 日志以时间戳命名，方便查找

开发者选项
----------

* 手机设备与电脑连到同一个网段内
* 打开本应用，将网络调试选中。
* 在电脑的执行adb connect ip就可以直接连到设备
